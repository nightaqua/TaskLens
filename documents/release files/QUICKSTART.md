# Quick Start Guide: TaskLens

## Installation

1. Install TaskLens (Community Plugins or manual).
2. Enable the plugin.
3. On first run, the Welcome tour appears.

## First Steps

1. Open the dashboard
   - Command: Open Dashboard (All-in-One)
   - Ribbon: Move icon
2. Arrange widgets
   - Unlock layout, drag widgets, then lock layout
3. Add tasks
   - Plus button in Task List header or Quick Add Task command

## Managing Tasks

- Edit: Pencil icon on a task
- Delete: Trash icon on a task
- Toggle: Checkbox
- Filter: Click any stat card (Urgent, Overdue, Completed, etc.)

## Task Format

- [ ] My task due:: 2024-05-20
- [ ] Long task start:: 2024-05-10 due:: 2024-05-20
